# 2-Dimensional Depth Krichhoff Migration Scripts
In this folder, different scripts are all used for do the 2-Dimensional Depth Krichhoff Migration.  
All scripts are second developed based on the orginal version written by DR Umberta Tinivella and DR Michela Giustiniani @OGS, ITALY:
https://github.com/utinivella/ISTRICI/blob/main/TRAD_V1/PerformPSDM  
To Perform the 2D Kirchhoff Migration, some SU (Seismic Unix) tools are used and the guide information for all these commands can checked.
The core commands are unif2, ray2d and sukdmig2d. The optinal tools are sudivcor and sustack.
